# Developer Hub

Welcome to the Developer Hub - a centralized knowledge base for Nexus development teams.

## Focus

This Dev Hub documents **technologies and patterns actually used in Nexus applications**, not every framework in existence. We focus on:

- **Java/Spring Boot** - Backend development
- **React/Vite** - Frontend development
- **Next.js** - Full-stack applications
- **PostgreSQL** - Database
- **AWS** - Cloud infrastructure
- **Vercel** - Frontend deployment (with BFF pattern)
- **Google OAuth** - Authentication

## Domains

Domain-specific documentation organized by area:

- **API Integration** - Backend (Java/Spring) and Frontend (React) patterns
- **Platform Engineering** - Deployment strategies (Vercel, AWS)
- **Security** - Security practices and procedures
- **Nexus Agents** - AI agent specifications and workflows

## Key Principles

1. **Technology Focus** - Document only technologies used in Nexus apps
2. **Practical Patterns** - Reusable architecture patterns with examples
3. **Read-Only Hub** - All content is publicly accessible, no authentication required
4. **Responsive Design** - Works on all devices (mobile, tablet, desktop)
